核心功能

顶栏中的功能按钮提供了完整的无障碍支持

🔍

智能缩放

动态调整页面内容大小，支持从50%到200%的缩放范围。实时显示当前缩放级别。

快捷键：

  • Ctrl/Cmd + + 放大
  • Ctrl/Cmd + - 缩小
  • Ctrl/Cmd + 0 重置
🖱️

鼠标增强

提供两种鼠标增强模式，帮助视力不佳的用户更好地定位和操作。

功能模式：

  • 大鼠标：放大鼠标光标至64x64像素
  • 十字线：显示红色定位十字线

两种模式可同时启用，提供双重定位辅助。

🔊

语音朗读

完整的文本转语音功能，支持中文朗读和多参数调节。

主要特性：

  • 语速调节（0.5x - 10x）
  • 音量控制
  • 悬停自动朗读
  • 行朗读模式
🌙

主题切换

在浅色和深色主题间无缝切换，适应不同环境和使用习惯。

主题特点：

  • 浅色模式：高对比度，适合白天使用
  • 深色模式：低亮度，减少眼睛疲劳

系统自动保存主题偏好，下次访问时自动应用。

无障碍设计原则

无障碍设计的核心是确保所有用户，无论其能力如何，都能平等地访问和使用数字内容。这包括视觉、听觉、运动和认知等各方面的考虑。

现代网页设计应遵循WCAG（Web内容无障碍指南）标准，确保内容的可感知性、可操作性、可理解性和鲁棒性。

技术实现

本页面使用纯JavaScript实现所有无障碍功能，不依赖任何外部库。功能包括：

  • 动态DOM操作
  • 本地存储状态管理
  • Web Speech API集成
  • CSS变量主题系统
  • 响应式布局设计

使用建议

为了获得最佳体验，建议尝试以下操作：

  1. 启用大鼠标功能，观察光标变化
  2. 打开十字线，移动鼠标查看定位效果
  3. 使用语音朗读，悬停在文字上体验
  4. 切换主题，感受不同视觉效果
  5. 使用Tab键测试键盘导航

集成指南

如何在您的网页中集成这些无障碍功能

🚀 快速开始

只需三个步骤即可在任何网页中启用所有无障碍功能：

  1. 复制 styles.css 和 script.js 到您的项目
  2. 创建您自己的 index.css 文件（见下方CSS配置指南）
  3. 在HTML中引入这三个文件并确保HTML结构正确
<head>
  <link rel="stylesheet" href="styles.css">
  <link rel="stylesheet" href="index.css"> <!-- 您创建的自定义样式 -->
</head>
<body>
  <header id="topbar" class="topbar">...</header>
  <main>...</main>
  <script src="script.js"></script>
</body>

📝 HTML结构要求

确保您的HTML包含以下必要的结构：

<header id="topbar" class="topbar">
  <div class="topbar-container">
    <h1 class="site-title">您的网站标题</h1>
    <nav id="navbar" class="navbar"></nav>
  </div>
</header>

<main>
  <!-- 您的主要内容 -->
</main>

🎨 CSS自定义选项

创建您自己的 index.css 文件，不要复制原始文件。新项目中只需配置以下内容：

/* 1. 修改全局CSS变量（可选） */
:root {
  --primary-color: #1a73e8;    /* 改为您的主题色 */
  --text-color: #202124;
  --background-color: #ffffff;
  --border-color: #dadce0;
  --hover-background: #f8f9fa;
}

/* 2. 添加您的页面自定义样式 */
.page-header {
    background: linear-gradient(135deg, #1a73e8 0%, #0d47a1 100%);
}

/* 3. 自定义内容区域样式 */
.main-content {
    max-width: 1200px;
    margin: 2rem auto;
    padding: 0 2rem;
}

/* 4. 自定义页脚 */
.footer {
    background-color: var(--hover-background);
    padding: 2rem;
    text-align: center;
}

关键点：

  • 修改 --primary-color 改变全站颜色主题
  • 修改 .page-header 的渐变色为您的品牌色
  • 调整 max-width 控制内容宽度
  • 定制页脚、标题等特定区域样式

⚙️ 功能配置修改

在 script.js 中修改以下配置对象：

// 缩放配置
const ZOOM_CONFIG = {
    min: 50,           // 最小缩放50%
    max: 200,          // 最大缩放200%
    default: 100,      // 默认100%
    step: 10           // 每次改变10%
};

// 语音配置
const SPEECH_CONFIG = {
    speedMin: 0.5,
    speedMax: 10,
    speedDefault: 1,
    lang: 'zh-CN'      // 支持其他语言代码
};

🔧 功能启用/禁用

编辑 NAVBAR_FEATURES 数组以启用或禁用特定功能：

// 要禁用某个功能，只需从数组中移除该对象
const NAVBAR_FEATURES = [
    // 注释掉或移除要禁用的功能
    // { id: 'speech-panel', ... },
    { id: 'zoom-controls', ... },
    { id: 'theme-control', ... }
    // 保留您需要的功能
];

📡 事件与API

在控制台中访问全局管理器对象：

// 语音控制
speechManager.speak("要朗读的文本");
speechManager.setSpeed(1.5);  // 设置语速
speechManager.setVolume(0.8); // 设置音量

// 缩放控制
zoomManager.applyZoom(150);   // 设置缩放比例

// 主题控制
themeManager.toggleTheme();   // 切换主题

// 行朗读
lineReaderManager.readNextLine();
lineReaderManager.readPreviousLine();

❓ 常见问题

Q: 语音功能在某些浏览器中不工作？
A: 请检查浏览器是否支持Web Speech API。某些浏览器可能需要启用相关权限。
Q: 如何改变主题颜色？
A: 修改CSS变量 --primary-color 和其他颜色变量。
Q: 能否在移动设备上使用？
A: 完全支持。所有功能都已针对触屏设备优化。
Q: 如何关闭某些功能？
A: 从 NAVBAR_FEATURES 数组中移除相应的功能配置。
